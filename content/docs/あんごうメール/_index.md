# あんごうメール

あんごうメール一覧です。

## あんごうメールの送り方
1. メニューから「メール」を選択
2. メール閲覧画面でXボタンを押してメール作成画面へ
3. 宛先はサテライト管理人（真ん中の紋章）を選択
4. プレゼントはつけなくても良いです
5. 件名に「あんごう」と入力する
6. 本文に暗号を入力する
7. 「そうしん」を選択


ちなみに最初から適合したところまでしか認識しない  

件名が「あんごうううう」だったり
本文の最後に余計な文字が付いていても問題なし
